# FitStart

FitStart is a Java-based fitness management web application developed using JSP, Servlets, MySQL, and Apache Tomcat.

## Features
- User Registration and Login
- Fitness tracking system
- Workout management
- User dashboard

## Technologies Used
- Java
- JSP / Servlets
- MySQL
- Apache Tomcat
- HTML, CSS, JavaScript

## How to Run
1. Import project in Eclipse/IntelliJ
2. Configure Apache Tomcat server
3. Setup MySQL database
4. Run on server

## Note
This is a college project for learning web development using Java.
